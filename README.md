# gravity
2D 2-Spieler Space-Shooter über mehrere Runden mit Tastatursteuerung für Browser

Die Spielidee kenne ich seit Anfang der 1990er. Ich hatte es damals auf dem 5MHz PC auf dem Hercules-Monochrom Bildschirm laufen.
Runden gab es damals nicht, aber Planet und Gravitation ließen sich einzeln zuschalten.
Es gab auch ein Energiesystem um passend zwischen Waffen und Schilden zu verteilen. - Habe ich hier weggelassen. Das besondere ist einfach die Vektorsteuerung. Jedes Manöver wird mit dem Haupttriebwerk durchgeführt nachdem das Schiff passend ausgerichtet ist. An der Tastatur super intuitiv.

Habe auch eine Maussteuerung eingebaut, damit du als Single-Player auf dem Handy spielen kannst. Ich sag mal: Viel Erfolg dabei.

Ach ja: Ich hatte das Spiel in den letzten Jahren schon 2x programmiert. 1x in Scratch. Das war angenehm einfach, auch wenn ich dort nicht die ganzen Feature reinfrickeln möchte die es hier im html gibt. Danach 1x in python in einer Datei ohne Bilder zu laden. Fühlte sich an wie assembler schreiben. Jede Linie selbst ziehen und unendliche koordinatentransformationen.

## Features

- Eine einzige HTML‑Datei
- Bedienung 2 Spieler per Tastatur oder Player vs. Computer per Maus
- Feature wie Planet mit Kollision, Gravitation, Asteroiden, Meteoritenschwärme 
- werden in den einzelnen Runden unterschiedlich kombiniert
- Startet mit PvP im Demo-Modus
